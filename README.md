一个简单的 Task 管理, 使用 Yammer 帐号登录

## Requirements

* ruby 2.0+
* Rails 4.0.0

## Getting Start

* Clone Resosotory, `git clone git@github.com:JustQyx/colla_board.git`
* run `bundle install`
* run `rake db:create` & `rake db:migrate`
* `rails s`
